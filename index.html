KTMouseGestures


Simple mouse gesture functions for google chrome.

——————————————
right button click and move
——————————————
U = UP
D = DOWN
L = LEFT
R = RIGHT

———————
normal gestures
———————
UD = reload
UDU = super reload
DR = close
L = back
R = forward
D = new tab
DU = new tab(background)

———————
link gestures
———————
link+D = new tab
link+DU = new tab(background)
